# canvas 2d使用及动画

## 主要内容
通过canvas画布实现2D动画基础，获取2D上下文绘制矩形，调整尺寸属性（里面特别强调了css对canvas的影响)
强调清除画布防残影


## 答疑
Q：通过css修改canvas大小和直接修改canvas大小有何不同？
A：
css修改只是改变了显示大小，canvas的实际绘制大小没有变，导致绘制内容被拉伸变形。
直接修改canvas的width和height属性才是真正改变了绘制大小。
